Talwind CSS Box Model Example

Box Model


This box demonstrates the properties of the box model using Tailwind CSS:


Padding

This box has p-4, which means padding of 1rem (16px).


Border

This Box has border-4 a dashed border with border-blue-400.


Margin

This box is seperated from other using mb-4(margin-bottom).


Width & Height

This box has w-64(16rem) and h-32 (8rem).
